# 🌟 Global Talent (全球之星)

> **"Built by freelancers, for freelancers."** — Managed by **TuanTuan Panda (团团熊猫)**

Global Talent is a production-grade, high-fidelity gig marketplace platform designed specifically for the Malaysian and global freelance community. From competitive gaming boosting to enterprise-level software development, Global Talent provides a secure, transparent, and aesthetically premium environment for talent transaction.

---

## ✨ Key Features

- **🛡️ 7% Platform Fee**: One of the lowest commission rates in the industry, ensuring creators keep more of what they earn.
- **🧾 Malaysia LHDN Compliant e-Invoice**: Automated generation of professional, tax-compliant receipts for every transaction (8% SST integrated).
- **💳 Multi-Channel Payment**: Seamless integration with **Stripe**, **TNG eWallet**, **Alipay**, and **Malaysian FPX Bank Transfer**.
- **💎 Glassmorphism UI/UX**: A 3A-grade aesthetic featuring vibrant gradients, subtle micro-animations, and high-fidelity interactive elements.
- **🚀 Dual-Role Synergy**: Switch between **Buyer** and **Seller** roles with a single click. Publish your own Gigs in seconds.
- **💻 Developer Portal**: Specialized "God Mode" dashboard for platform owner **TuanTuan Panda** to monitor system health and GMV in real-time.

---

## 🛠️ Technology Stack

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS with modern Glassmorphism tokens
- **Backend/Hosting**: [Firebase](https://firebase.google.com/)
- **Payments**: [Stripe SDK](https://stripe.com/)

---

## 🎨 Design System

Our design philosophy centers on **"Premium Clarity"**. 
- **Colors**: Deep Midnight (#0f172a) combined with Indigo-Pink gradients.
- **Effects**: Backdrop blur (blur-20), multi-layered shadows, and pulsing border glows.
- **Typography**: Inter / System sans-serif for maximum readability across devices.

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/TuanTuanPanda/Global-Talent.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Deploy to Firebase**
   ```bash
   firebase deploy
   ```
